# .ai 文件夹说明

本文件夹用于记录 AI 辅助开发过程中的**提示词输入**和**关键决策**。

## 为什么需要这个文件夹？

- 避免重复向 AI 描述同一个问题
- 追溯代码变更的“为什么”（配合 Git commit）
- 团队共享有效的提示词，提升协作效率
- 评估不同提示词的效果，持续优化


## 文件夹结构

.ai/
├── README.md # 本文件
├── prompts/ # 提示词输入记录
│ ├── 2026-01-15-账户功能.md
│ └── ...
└── outputs/ # （可选）AI 输出的关键内容摘要
└── ...

## 文件说明
- `README.md`: 本文件，包含文件夹结构说明、为什么需要这个文件夹等。
- `prompts/`: 提示词输入记录目录，每个文件对应一个日期，记录该日期的提示词输入。
- `outputs/`: （可选）AI 输出的关键内容摘要目录，每个文件对应一个日期，记录该日期的 AI 输出摘要。
- 同一个文件代表同一个对话里的内容，可以根据上下文进行关联。
