**5个默认账户**：
- 现金（visible=true, isDefault=true）
- 投资（visible=true）
- 贷款（visible=true）
- 收入（visible=false，系统隐藏）
- 支出（visible=false，系统隐藏）

**6种交易类型**：
| 类型 | 分录规则 |
|------|----------|
| 收入 | 借:现金 贷:收入 |
| 支出 | 借:支出 贷:现金 |
| 投资 | 借:投资 贷:现金 |
| 投资到期 | 借:现金 贷:投资(本金) + 借:现金 贷:收入(利息) |
| 贷款到账 | 借:现金 贷:贷款 |
| 还贷 | 借:贷款 贷:现金(本金) + 借:支出 贷:现金(利息) |
